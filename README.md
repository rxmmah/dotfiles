# dotfiles for the minimal mind
wayland and dwl on Arch 'btw'

## main apps

window manager: dwl ( with: dwlb, someblocks)

terminal: foot & tmux

shell: zsh with starship

music player: cmus

file manager: lf

video player: mpv with uosc

rss reader: newsboat

editor: nvim

pdf and book reader: zathura
